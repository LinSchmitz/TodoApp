# Todo App

A simple todo list app built with Vanilla JavaScript to practice core JS concepts.

## What I'm learning

- DOM manipulation
- Event listeners
- localStorage
- Array and object methods

## Features

- Add new tasks
- Mark tasks as complete
- Delete tasks
- Filter tasks (all / active / completed)
- Data persists after page refresh (localStorage)

## Tech stack

- HTML
- CSS
- JavaScript (no frameworks)

## How to run

1. Clone or download the project
2. Open `index.html` in your browser
3. No install needed

## Project structure

```
todo-app/
├── index.html
├── style.css
└── app.js
```
